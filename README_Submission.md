# Care4Child - منصة رعاية الأطفال وإدارة اللقاحات

هذا المشروع مجهز بالكامل ليعمل باستخدام بيئة Docker لضمان سهولة التشغيل والمطابقة التامة لبيئة التطوير بدون الحاجة لتثبيت أي متطلبات برمجية إضافية مثل Python أو مكتبات أخرى.

## متطلبات التشغيل
1. تثبيت برنامج **Docker Desktop** (أو Docker Engine).
2. التأكد من تشغيل خدمة Docker في الخلفية.

## خطوات التشغيل
1. افتح موجه الأوامر (CMD) أو (Terminal) داخل هذا المجلد (المجلد الذي يحتوي على ملف `docker-compose.yml`).
2. قم بتنفيذ الأمر التالي لإنشاء وتشغيل الحاويات:
   ```bash
   docker-compose up --build
   ```
3. انتظر حتى تكتمل عملية تحميل المكتبات وبدء تشغيل السيرفر.
4. افتح المتصفح وادخل على الرابط التالي:
   http://localhost:8000

## بيانات الدخول للتجربة
قاعدة البيانات (`db.sqlite3`) المرفقة تحتوي على بيانات تجريبية جاهزة للاستخدام:

- **حساب الوزارة (المشرف العام):**
  - اسم المستخدم: `ministry` (أو الإيميل الخاص بها)
  - كلمة المرور: (الرجاء استخدام الكلمة التي تم إعدادها، أو يمكننا إضافة حسابات محددة هنا)
  
- **حساب أحد المراكز الصحية (لتجربة تسجيل الأطفال):**
  - اسم المستخدم: `center1`
  - كلمة المرور: `(كلمة المرور الخاصة بالمركز)`

*(ملاحظة للطالب: قم بتعديل بيانات الدخول أعلاه لتطابق الحسابات الموجودة فعلياً في قاعدة البيانات الخاصة بك)*

## مكونات النظام المرفقة
- **النظام الخلفي (Backend):** مبني باستخدام إطار عمل Django 5.0.
- **قاعدة البيانات:** SQLite (مرفقة ومليئة بالبيانات).
- **ملف خدمة الإشعارات:** `serviceAccountKey.json` للربط مع Firebase Cloud Messaging.
- **التصميم المتجاوب:** يدعم كافة الشاشات وأجهزة الجوال (Responsive Design).
